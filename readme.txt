
React Weather App
A simple React.js application that fetches and displays current weather data for a specified city using the OpenWeatherMap API.

Replace the placeholder API key in App.tsx with your OpenWeatherMap API key:

tsx
Copy code
const API_KEY = "6e84e57a336fa60ab09c4e1008c62e38";

bash
Copy code
npm start
Open in Browser:

Visit http://localhost:3000 in your browser.
Fetch Weather Data:

Enter a city name, click "Get Weather" to fetch and display weather information.
